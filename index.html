Основные свойства логарифмов

  • Свойство 1: logₐ(x * y) = logₐ(x) + logₐ(y) — логарифм произведения равен сумме логарифмов.
  • Свойство 2: logₐ(x / y) = logₐ(x) - logₐ(y) — логарифм частного равен разности логарифмов.
  • Свойство 3: logₐ(x^n) = n * logₐ(x) — логарифм степени равен произведению степени на логарифм основания.
  • Свойство 4: logₐ(a) = 1 — логарифм числа по его собственному основанию равен 1.
  • Свойство 5: logₐ(1) = 0 — логарифм числа 1 по любому основанию равен 0.
  • Свойство 6: loga (bc · d) = loga (bc) + loga d
  • Свойство 7: loga (bc) = c · loga b
  • Свойство 8: loga (x) = logb (x) / logb (a)
  • Свойство 9: alogb c = clogb a
  • Свойство 10: loga b = logd b / logd a = 1 / logb a
  • Свойство 11: loga an bn = loga b
  • Свойство 12: loga ak bn = n/k · loga b
  • Свойство 13: loga ak = 1/k · loga b

Примеры использования логарифмов

Пример 1: log₂(8) = 3 — так как 2^3 = 8.

Пример 2: log₁₀(1000) = 3 — так как 10^3 = 1000.
